## Sintaxes

- If/else -> comando condicional

    ```
    Exemplo 1
    //Num sendo o valor 0

    let num = 0

    if(num>5){
        return true
    } else {
        return false
    }

    //Retornará false, pois 0 não é maior que 5

    Exemplo 2

    //Num sendo o valor 10

    let num = 10

    if(num>5){
        return true
    } else {
        return false
    }

    //Retornará true, pois 10 é maior que 5
    ```

- Sinal "!=" é sinal de diferente "≠"
- Sinal "&&" é basicamente um "and"
