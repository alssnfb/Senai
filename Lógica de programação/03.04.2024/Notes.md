## Laços de Repetição

```
    while (condição) -> Estrutura de repetição ENQUANTO, a condição for verdadeira
```        
### Exemplo While

```
    let contador = 0
    while(contador < 10){
        alert("alguma coisa")
    }
```    
- Atalhos    
    > - Contador ++ -> (Contador = Contador + 1)
    > - Variavel ++ (Incremento) -> + 1
    > - Variavel -- (Decremento) -> - 1
